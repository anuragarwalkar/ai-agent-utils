Privacy Policy

Last updated: September 11, 2025

AI Agent Utils does not collect, store, or share any personal information from users.

What the Extension Does

  • All configuration (such as API keys, model name, and temperature settings) is stored locally in Chrome extension storage on your device.
  • Page content is only processed when you explicitly activate a tool (e.g., Text Fixer, Web Reader, Question Answerer, Input Filler).
  • If you configure an external AI API (such as OpenAI, Anthropic, or a local API-compatible server), the selected text will be sent directly to that API for processing. AI Agent Utils does not intercept, log, or store this data.
  • The extension does not track browsing history and does not run automatically on all sites. It only operates on the active tab when you choose to use it.

Data Sharing

We do not sell, rent, or share user data with any third parties. All processing happens locally or via the AI API you configure.
